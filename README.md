# coding-mate
## 记录编码时光！ Here always with you:wink:
A Windows desktop app project in C# VS2015.

=====

### 更新备忘：

### 使用说明
1. 在执行文件同级目录下 **提示音频.mp3**  为到点提醒音频
2. **codeRecord.xml** 为记录文件，存储着你的数据，请勿编辑！
